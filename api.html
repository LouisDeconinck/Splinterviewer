Splinterviewer

Unofficial Splinterlands API Documentation

Last updated: 4 October 2021

As there does not exist official documentation for the Splinterlands API, I've bundled all available information here. This should help you get started with your own Splinterlands projects.

Help keep this information up to date and consider donating if it has helped you. You can send DEC, SPS, SPT, Hive, HBD, or cards to @solaito. You can also use the buttons at the bottom of the page to securely donate through Hive Keychain.

API endpoint: https://api2.splinterlands.com/purchases/check_code?code=K5XF-7N1N

Currency transfer

{"token":"SPS","qty":300,"to":"sl-hive","memo":"smitop","app":"splinterlands/0.7.139","n":"CaVA9uX3DF"}

This is the format needed when sending a currency transfer. The n value is a random number used to check for duplicate transactions, you can just omit it.
